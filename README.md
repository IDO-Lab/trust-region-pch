# PCH (Projection Convex Hull)
This repository provides a self-contained implementation of **PCH** and a runnable demo on a synthetic **Polyhedron** benchmark.

## Requirements
- Python **3.9+** (required by dictionary merge / modern tooling)
- Dependencies: `numpy`, `scipy`

Install:
```bash
pip install -r requirements.txt
```

## Quickstart
Run:
```bash
python scripts/run_pch_polyhedron.py --beta 4 --shift_th 0.03 --learning_rate 0.1 --weight_lr 0.0001
```

Outputs:
- Synthetic datasets: `Data/polyhedron15_{dim}.csv`
- Results table: `results/pch_polyhedron.csv`

## Customize
You can change dimensions and sample size:
```bash
python scripts/run_pch_polyhedron.py --dims 2 3 4 5 6 7 8 10 --n_samples 20000
```

## Repository Layout
```text
pch/                    PCH library code
  pch.py                Core implementation
scripts/
  gen_polyhedron_data.py    Synthetic benchmark generator
  run_pch_polyhedron.py     Run PCH and export a CSV table
results/
  pch_polyhedron.csv     (generated)
Data/
  polyhedron15_*.csv     (generated)
```

## Notes
- The synthetic benchmark is generated by sampling uniformly from `[-1,1]^d` and labeling points by membership in a randomly generated polyhedron (intersection of halfspaces).
